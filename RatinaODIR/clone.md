## 📂 Dataset

This project uses the **ODIR (Ocular Disease Intelligent Recognition) dataset**, which contains retinal fundus images labeled with multiple eye diseases.

### Dataset Details:
- Retinal fundus images of both left and right eyes  
- Multi-label annotations (one image can have multiple diseases)  
- Includes structured annotation file (CSV/Excel format)  

### Disease Classes:
- N – Normal  
- D – Diabetes  
- G – Glaucoma  
- C – Cataract  
- A – Age-related Macular Degeneration  
- H – Hypertension  
- M – Pathological Myopia  
- O – Other Abnormalities  

### Key Characteristics:
- Real-world medical dataset with multiple co-existing conditions  
- Designed for multi-label classification tasks  
- Class imbalance present (some diseases are underrepresented)
